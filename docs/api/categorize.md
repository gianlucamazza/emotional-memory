# Categorize

Discrete emotion categorization via Plutchik's wheel (Plutchik, 1980).

Maps continuous (valence, arousal) coordinates from the Russell circumplex to one of 8 primary
emotions with three intensity tiers. An optional dominance value disambiguates the fear/anger
region following the PAD model (Mehrabian & Russell, 1974).
